Every decision should move the platform closer to becoming the trusted place where people learn, build, deploy, buy, sell, and scale AI-powered systems.

## Purpose

This directory is the **brand operations home** for Melanated In Tech. Every visual asset, copy template, voice guideline, and brand standard lives here. Any agent or team member producing something visible to the public must reference this directory first.

The brand is not just aesthetics. It is the primary competitive advantage that no well-funded competitor can copy: genuine cultural authority, technical credibility, and an unambiguous identity built for a specific community.

### Color System

```css
/* Primary palette */
--mit-black:     #0A0A0A;   /* Page backgrounds */
--mit-dark:      #111111;   /* Elevated surfaces, cards */
--mit-charcoal:  #1A1A1A;   /* Inputs, borders */
--mit-green:     #00FF41;   /* CTAs, highlights, active states — THE signature color */
--mit-green-dark:#00CC33;   /* Hover states */

/* Text */
--mit-white:     #FFFFFF;   /* Primary text */
--mit-gray:      #A0A0A0;   /* Secondary text */
--mit-muted:     #666666;   /* Disabled, placeholders */

/* Status */
--mit-danger:    #FF4444;
--mit-warning:   #FFB800;
--mit-success:   #00FF41;   /* Same as accent green */
```

**The #1 rule:** Neon green `#00FF41` ONLY on dark backgrounds. Never on light.
**The #2 rule:** Green is reserved for CTAs and highlights. Overuse kills the effect.

## Brand Red Lines (Never Cross These)

```
✗ Corporate-speak or DEI buzzwords
✗ "Unlock your potential" / "Transform your career" / "Join the revolution"
✗ Passive voice when active voice works
✗ Overpromising outcomes (income, time-to-results)
✗ Auto-publishing AI content without human review
✗ Fake social proof before it's earned
✗ Simplifying AI concepts to the point of inaccuracy
✗ Neon green on light backgrounds
✗ Mixing brand fonts with web fonts
```
