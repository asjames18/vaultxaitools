## Use Case Page Template

Every use-case page follows this structure:

```
/[use-case-slug]  OR  /solutions/[use-case-slug]

1. Hero (H1 + subheadline + primary CTA)
2. Problem Section (pain points, in bold)
3. Solution Overview (what MIT provides)
4. How It Works (3-step visual)
5. Products/Tools Featured (with links to marketplace)
6. Case Study or Social Proof
7. FAQ (3-5 questions — triggers FAQPage schema)
8. CTA Section (primary conversion action)
```
