## SUBJECT LINE OPTIONS (write 3 for A/B testing)

**Formula options:**
- Specificity + Curiosity: "The MCP pattern that saved me 40 hours"
- Result + Tool: "How to build a customer support agent in 3 hours"
- Observation + Stakes: "AI agents are eating SaaS. Here's who's winning."
- Direct + Useful: "Step-by-step: Local AI setup on any machine"

**Never:**
- "This week's newsletter" ← boring
- "Exciting news from MIT!" ← corporate
- "You won't believe..." ← clickbait

**Write 3 subject line options:**
```
Option A: [subject line — specificity/result]
Option B: [subject line — curiosity/stakes]
Option C: [subject line — direct/useful]
```
