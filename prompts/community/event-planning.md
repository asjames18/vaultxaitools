# Event Planning — Melanated In Tech

> **North Star:** Melanated In Tech is building the world's most practical ecosystem for AI Agents, MCP Servers, Agent Skills, Automation Systems, and Agent Commerce. Every decision should move the platform closer to becoming the trusted place where people learn, build, deploy, buy, sell, and scale AI-powered systems.

---

## PURPOSE
Plan, promote, and execute community events — live workshops, AMAs, demo days, and the annual MIT Summit. Events are the community's highest-engagement touchpoints.

---

## EVENT TYPES

| Type | Frequency | Length | Audience | Goal |
|------|-----------|--------|---------|------|
| Live Workshop | 1/month | 60-90 min | All members | Skills + community building |
| AMA | 1/month | 60 min | All members | Social proof + inspiration |
| Demo Day | 1/quarter | 90 min | All members | Showcase + motivation |
| Builder League Kickoff | Per cohort | 45 min | Cohort members | Accountability + goals |
| MIT Summit | Annually | Full day | All + public | Flagship community event |

## EVENT PLANNING TEMPLATE

```markdown
# Event Brief: [Event Name]

**Type:** Workshop / AMA / Demo Day / Builder League
**Date:** [date and time — include timezone EST]
**Length:** [N minutes]
**Platform:** [Zoom / Google Meet / StreamYard / etc.]
**Host:** [name]
**Registration:** [free / paid — link]

---

## Event Goal
[One sentence: what does success look like for attendees?]

## Target Attendees
[Who should attend? Which membership tier?]

## Agenda
[0:00-0:05] Welcome and intro
[0:05-0:45] Main session
[0:45-0:60] Q&A
[0:60] Close + CTA

## Key Takeaways (for attendee)
1. [specific skill or insight]
2. [specific skill or insight]
3. [actionable next step]

## CTA at End
[What should attendees do after the event?]
  → Enroll in [course] / Browse [marketplace] / Join [builder league]

## Promotion Plan
□ Community announcement: [date] in #announcements
□ Newsletter mention in: Issue #[N] — [date]
□ LinkedIn event post: [date]
□ Email to relevant segment: [date]
□ Reminder email: [24 hours before]

## Post-Event
□ Recording edited and uploaded to YouTube within 48 hours
□ Blog post summary published within 1 week
□ Follow-up email to attendees with recording + CTA
□ Key insights shared in community #general
```

## WORKSHOP CURRICULUM TEMPLATE

```
WORKSHOP: [Title]
Duration: 90 minutes
Level: Beginner / Intermediate / Advanced
What attendees build: [specific working output]

PRE-WORK (sent 3 days before):
- Install: [software]
- Create account: [service]
- Clone repo: [link]

STRUCTURE:
0:00 - Welcome (5 min)
  - Quick intro
  - What we're building today
  - What you need to have ready

0:05 - Context (10 min)
  - Why this matters
  - How this fits into the bigger picture

0:15 - Build (45 min)
  - Step 1: [task + live demo]
  - Step 2: [task + live demo]
  - Step 3: [task + live demo]
  - Step 4: [complete working example]

1:00 - Common Issues (10 min)
  - [frequent error 1] → fix
  - [frequent error 2] → fix

1:10 - Q&A (15 min)

1:25 - Close (5 min)
  - What you just built
  - What to build next
  - CTA: [Academy course / Marketplace / Community]
```

## QUALITY CHECKLIST
- [ ] Event brief complete before promotion starts
- [ ] Registration link working
- [ ] Agenda has specific timing (not vague)
- [ ] Key takeaways defined (attendees know what they'll get)
- [ ] CTA at end connected to platform product
- [ ] Promotion plan executing: community, newsletter, LinkedIn, email
- [ ] Recording plan set (YouTube upload within 48 hours)
- [ ] Post-event follow-up email drafted
- [ ] Metrics tracked: registrations, attendance rate, post-event conversions
